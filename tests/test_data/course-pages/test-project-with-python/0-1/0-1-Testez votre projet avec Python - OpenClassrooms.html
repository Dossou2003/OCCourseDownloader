Introduction du cours

 

Avez-vous déjà passé plusieurs heures à chercher l'origine d'un bogue ? Vous testez, testez et retestez manuellement avant de trouver enfin ce qui posait problème ! Je suis sûre que cette situation vous parle ! Et si je vous disais que vous auriez pu automatiser ces tests en laissant l'ordinateur jouer le détective à votre place ?

Python vous permet de tester très facilement un projet. Découvrez dans ce cours la puissance des tests unitaires ainsi que les différentes formes de tests. Vous verrez également comment devenir plus efficace utilisant le test-driven development ! Vous verrez, vous serez impressionné·e !

Alors, prêt·e ? Inscrivez-vous !

Prérequis :

Ce cours étant basé sur Python, il est nécessaire d'en connaître la syntaxe et d'être capable d'utiliser les lignes de commande. Ce n'est pas encore le cas ? Commencez par suivre le cours Démarrez votre projet avec Python puis le cours Découvrez la Programmation Orientée Objet avec Python !

Il est également nécessaire d'être à l'aise avec Git et de connaître le processus de pull request.

A l'issue de ce cours vous aurez développé les compétences suivantes :

  • Utiliser le test-driven development pour ajouter une nouvelle fonctionnalité à un projet.

  • Tester efficacement un projet en Python.

  • Connaître la librairie Pytest

  • Organiser des tests en suivant une approche orientée objet.

  • Utiliser un mock pour imiter le résultat d'une requête HTTP.
